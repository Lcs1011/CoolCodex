### 支持的CMD指令

cbot c
执行剪切板的内容 c 代表 clipboard

cbot 回车
提示继续按回车执行 剪切板的内容

cbot d

如果没有设置efautl 路径 就提示找不到 无法执行

`cbot <执行文件的绝对路径径>`

Rust 程序会自动生成两个运行文件：

```
CBotNetLinkerState.txtCBotNetLinkerLog.md
```

作用：

```
CBotNetLinkerState.txt记录最后处理过的 Telegram update_id，防止重启后重复执行旧消息。CBotNetLinkerLog.md记录收到的消息、执行的 cbot 命令、执行结果。
```
